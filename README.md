## できる事<br>
ver1<br>

・瞬発力トレーニング　制限時間：1秒<br>

## 工夫した点<br>
ver1<br>
・解答ボタン押した時、結果画面だけでなく次の問題も表示・選択できることでスピード感アップ！<br>
　瞬発力を鍛えるものなので、スピードが速くなったのはとても良かったかと思っています。<br>
ver2<br>
・問題＆答え＆コメントを追加機能をつけ、トレーニング！<br>
　頭の中を整理する機会づくりにもピッタリ♪<br>
・選択肢の４つは【時間軸（過去、将来）×行動軸（ブレーキ、アクセル）】という意味付けがあります。<br>
　そのため、これ以上選択肢を増やさなくても、シンプルに選択できると考えています。<br>

## 苦労や学び<br>
ver1<br>
・メモアプリ制作でDAO記述について処理の手順等は理解が深まっていたが、<br>
　何も見ないで打つことはできませんでした。身についていないことを痛感しました。<br>
・コントローラ・モデル・ビューや、クラス・メソッドをきちんと役割で分けて、<br>
　作成することを意識しました。<br>
　前回制作したメモアプリよりは、かなり見やすくなり無駄なものが少なくなったと感じました。<br>
　（今回は機能がまだ少ない分、そう感じたのかもしれませんが）<br>
・また機能を増やしていく予定です！<br>
ver2<br>
・１分経過後に画面を変える処理が難しかったです。<br>
　スタートボタン押してから現在時間取得・終了時間設定→セッションスコープへ<br>
　JavaScriptで１秒ごとに現在時間をチェックし、終了時刻になるとサーブレットへ<br>
　ランク判定してセッションスコープへ→ランク結果画面を表示<br>
　調べて試行錯誤してようやくできました。<br>
